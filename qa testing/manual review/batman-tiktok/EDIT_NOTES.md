# Batman Entry — TikTok Creative Cut

## Deliverables

- `batman_entry_tiktok_edit.mp4` — final 1080×1920 TikTok export
- `batman_entry_tiktok_edit.rushframe` — editable Rushframe project
- `analysis-core/` — local transcript, scene, moment, and media analysis used for the cut

## Creative structure

1. Cold-open on the payoff: “I brought justice to the table.”
2. Flash/glitch rewind into “My turn to deal.”
3. Fast escalation through the chair confrontation and repeated “calm down” reactions.
4. Batman reveal and “Batman in the building.”
5. Full justice payoff, followed by “Go fight them for me.”

## Edit treatment

- 22.40-second runtime
- 1080×1920, 30 fps, H.264/AAC
- Original stereo dialogue preserved
- Full 16:9 table shot retained inside a vertical dark-gradient layout so no entrant is cropped out
- Subtle animated punch-ins on every story beat
- Short flash/glitch impact layers at the cold open, rewind, reveal, and payoff
- Burned-in uppercase captions with black outline/shadow
- Yellow emphasis for Batman/payoff beats and blue emphasis for setup/reaction beats
- Profanity visually censored for a more platform-friendly upload
- TikTok and title-safe layout guides stored in the editable project

## Source-analysis notes

The source is a 31.37-second 1280×720 H.264/AAC clip. Local Rushframe intelligence produced 15 transcript segments. Optional Gemini visual analysis could not authenticate, and optional OCR/audio-event startup encountered a package entry-point collision, so the edit used the successful local transcript/media analysis plus direct frame review.
